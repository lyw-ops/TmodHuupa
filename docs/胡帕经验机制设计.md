# 胡帕经验机制设计文档

## 1. 核心定位

胡帕不是普通召唤宠物，而是整个 Mod 的核心伙伴与系统入口。

它的经验机制不应该只是简单地提升攻击力，而应该围绕以下目标设计：

1. 修复胡帕破碎的圆环；
2. 提高胡帕与泰拉瑞亚世界的同步程度；
3. 解锁更多宝可梦召唤权限；
4. 推进大胡帕追杀主线；
5. 最终解锁与大胡帕的决战。

核心设定可以概括为：

> 胡帕通过玩家的战斗、探索和 Boss 击杀吸收泰拉瑞亚世界的能量，逐步修复圆环，恢复召唤宝可梦和操控空间的能力。

胡帕不加入受伤、死亡或虚弱机制。它始终作为玩家的核心伙伴存在，成长压力主要来自等级上限、突破材料、剧情进度和 Boss 首杀。

## 2. 双经验系统

建议胡帕拥有两套成长经验：

| 系统 | 名称 | 来源 | 作用 |
|---|---|---|---|
| 主成长 | 圆环同步经验 | 打怪、Boss、事件 | 提升圆环同步等级 |
| 剧情成长 | 羁绊经验 | 主线、Boss 首杀、任务 | 推进剧情和最终战 |

简单来说：

```text
圆环同步经验 = 通过战斗成长
羁绊经验 = 通过剧情成长
```

这样可以防止玩家单纯刷怪把胡帕养满，而不推进游戏流程。

## 3. 圆环同步等级

胡帕的主等级不建议直接叫“等级”，而是叫：

> 圆环同步等级 Ring Sync Level

### 设计理由

胡帕来自宝可梦世界，刚进入泰拉瑞亚时力量不稳定。  
它需要通过战斗和材料，让自己的圆环逐渐适应泰拉瑞亚世界的法则。

## 4. 等级上限与阶段划分

胡帕最高等级为 Lv.100。  
每个阶段都有等级上限，需要突破后才能继续升级。

| 阶段 | 等级上限 | 游戏进度 | 胡帕状态 |
|---|---:|---|---|
| 破碎圆环 | Lv.10 | 开局 | 只能召唤弱小宝可梦 |
| 初步同步 | Lv.20 | 克苏鲁之眼后 | 圆环可以稳定开启 |
| 异界适应 | Lv.30 | 腐化/猩红 Boss 后 | 可召唤属性宝可梦 |
| 地狱共鸣 | Lv.40 | 地牢/地狱阶段 | 圆环可承受高能量 |
| 世界撕裂 | Lv.50 | 血肉墙后 | 大胡帕开始追踪 |
| 硬化圆环 | Lv.60 | 困难模式初期 | 可召唤困难模式宝可梦 |
| 星界感应 | Lv.70 | 机械三王后 | 可召唤强力宝可梦 |
| 生命觉醒 | Lv.80 | 世纪之花后 | 胡帕获得强辅助能力 |
| 天穹裂环 | Lv.90 | 石巨人后 | 可召唤神兽投影 |
| 完整圆环 | Lv.100 | 月亮领主后 | 解锁最终大胡帕战 |

## 5. 胡帕经验来源

胡帕的经验来源主要分为三类：

1. 普通怪物经验；
2. Boss 经验；
3. 圆环裂隙事件经验。

### 5.1 普通怪物经验

普通怪物给胡帕提供圆环同步经验。

| 怪物阶段 | 推荐经验范围 |
|---|---:|
| 开局地表怪 | 2-5 |
| 洞穴/沙漠/雪地怪 | 5-12 |
| 腐化/猩红/丛林怪 | 12-25 |
| 地牢/地狱怪 | 25-50 |
| 困难模式初期怪 | 60-120 |
| 机械 Boss 后怪 | 120-220 |
| 世纪之花后怪 | 220-400 |
| 月亮事件怪 | 500-900 |

### 5.2 Boss 经验

Boss 经验应该明显高于普通怪，但重复刷 Boss 的收益要降低。

| Boss 类型 | 首杀经验 | 重复击杀经验 |
|---|---:|---:|
| 史莱姆王 / 克苏鲁之眼 | 中量 | 30% |
| 世界吞噬怪 / 克苏鲁之脑 | 大量 | 30% |
| 蜂王 / 骷髅王 | 大量 | 30% |
| 血肉墙 | 阶段突破级 | 20% |
| 机械三王 | 大量 + 羁绊经验 | 20% |
| 世纪之花 | 大量 + 剧情解锁 | 20% |
| 月亮领主 | 终极突破经验 | 10% |

Boss 首杀不只是经验奖励，也应该触发胡帕剧情。

例如：

> 第一次击败血肉墙后，世界边界被撕开，大胡帕感应到了小胡帕的位置。

### 5.3 圆环裂隙事件经验

圆环裂隙是 Mod 的特色事件。

事件名称建议：

> 圆环裂隙 Ring Rift

触发条件可以是：

- 胡帕达到当前等级上限；
- 玩家击败关键 Boss；
- 特定环境随机出现；
- 大胡帕追踪剧情推进后出现。

完成事件后奖励：

| 奖励 | 用途 |
|---|---|
| 圆环同步经验 | 提升胡帕等级 |
| 羁绊经验 | 推进剧情 |
| 圆环碎片 | 制作突破道具 |
| 异界能量 | 召唤宝可梦 |
| 宝可梦契约材料 | 解锁新宝可梦 |

## 6. 经验计算方式

推荐经验公式：

```text
获得经验 = 怪物基础经验 x 进度倍率 x 参与倍率 x 等级压制倍率
```

其中：

| 参数 | 作用 |
|---|---|
| 怪物基础经验 | 由怪物阶段决定 |
| 进度倍率 | 根据当前世界进度调整 |
| 参与倍率 | 根据胡帕/宝可梦是否参与战斗调整 |
| 等级压制倍率 | 防止刷低级怪升级 |

## 7. 等级压制机制

为了防止玩家后期刷低级怪升级，需要经验衰减。

| 怪物阶段与胡帕阶段对比 | 经验倍率 |
|---|---:|
| 怪物高 2 阶段 | 150% |
| 怪物高 1 阶段 | 120% |
| 同阶段 | 100% |
| 怪物低 1 阶段 | 40% |
| 怪物低 2 阶段 | 10% |
| 怪物低 3 阶段以上 | 0% |

示例：

- Lv.8 胡帕打史莱姆：正常获得经验；
- Lv.60 胡帕打史莱姆：几乎没有经验；
- Lv.60 胡帕打困难模式怪：正常获得经验。

## 8. 突破机制

胡帕每 10 级会遇到一次等级上限。  
达到上限后，经验可以继续累积，但不能升级。  
玩家需要制作对应阶段的“圆环核心”进行突破。

### 8.1 突破材料表

| 突破 | 等级上限变化 | 道具名 | 主要材料 |
|---|---:|---|---|
| 初级突破 | 10 -> 20 | 粗糙圆环核心 | 铁/铅锭、凝胶、坠落之星 |
| 二阶突破 | 20 -> 30 | 辉光圆环核心 | 银/钨、金/铂金、宝石 |
| 三阶突破 | 30 -> 40 | 异化圆环核心 | 魔矿/猩红矿、暗影鳞片/组织样本 |
| 四阶突破 | 40 -> 50 | 熔火圆环核心 | 狱石锭、黑曜石、火焰花 |
| 五阶突破 | 50 -> 60 | 硬化圆环核心 | 钴蓝/钯金、光明/暗影之魂 |
| 六阶突破 | 60 -> 70 | 魔导圆环核心 | 秘银/山铜、视域/力量/恐惧之魂 |
| 七阶突破 | 70 -> 80 | 泰坦圆环核心 | 精金/钛金、神圣锭 |
| 八阶突破 | 80 -> 90 | 森灵圆环核心 | 叶绿锭、生命果、神庙材料 |
| 九阶突破 | 90 -> 100 | 星界圆环核心 | 四柱碎片、夜明锭 |

### 8.2 突破的剧情解释

突破不是单纯变强，而是：

> 胡帕的圆环与泰拉瑞亚世界的法则进一步同步。

不同阶段的矿石相当于“世界锚点”。

| 材料阶段 | 剧情含义 |
|---|---|
| 铁/铅 | 基础物质锚定 |
| 金/铂金 | 稳定圆环结构 |
| 魔矿/猩红矿 | 抵抗邪恶污染 |
| 狱石 | 承受地狱能量 |
| 困难矿 | 适应困难模式世界法则 |
| 叶绿锭 | 与生命能量同步 |
| 四柱碎片 | 接触宇宙级力量 |
| 夜明锭 | 完整圆环最终稳定 |

## 9. 升级奖励设计

胡帕每次升级不应该只加伤害。  
它的成长应该主要体现在功能解锁上。

### 9.1 召唤容量

| 胡帕等级 | 可出战宝可梦数量 |
|---|---:|
| Lv.1 | 1 只 |
| Lv.20 | 1 只 + 备用位 |
| Lv.40 | 2 只 |
| Lv.60 | 2 只 + 支援位 |
| Lv.80 | 3 只 |
| Lv.100 | 3 只 + 神兽支援位 |

说明：

- 出战宝可梦是常驻战斗伙伴；
- 备用位可以获得少量经验；
- 支援位用于短时间召唤；
- 神兽支援位不建议常驻。

### 9.2 宝可梦召唤池

| 胡帕等级 | 解锁召唤池 |
|---|---|
| Lv.1-20 | 初始宝可梦、小型宝可梦 |
| Lv.20-40 | 一阶进化、属性宝可梦 |
| Lv.40-60 | 二阶进化、特殊宝可梦 |
| Lv.60-80 | 强力宝可梦、准神 |
| Lv.80-100 | 神兽投影、终极技能 |

示例：

| 等级 | 解锁内容 |
|---|---|
| Lv.10 | 皮卡丘、小火龙、杰尼龟、妙蛙种子 |
| Lv.30 | 耿鬼、伊布进化、飞行系宝可梦 |
| Lv.60 | 路卡利欧、沙奈朵、暴飞龙 |
| Lv.80 | 烈咬陆鲨、班基拉斯 |
| Lv.90 | 裂空座投影 |
| Lv.100 | 大胡帕最终战 |

### 9.3 胡帕自身技能

胡帕本体建议走辅助路线，不建议做成强输出角色。

| 等级 | 技能 | 效果 |
|---|---|---|
| Lv.1 | 小圆环 | 偶尔发射小型能量弹 |
| Lv.10 | 圆环拾取 | 自动吸取附近特殊掉落 |
| Lv.20 | 空间闪避 | 玩家满血受伤时自动闪避一次，随后进入冷却 |
| Lv.30 | 异界牵引 / 属性拳 | 召出牵引圆环，并可近身打出火焰拳、冰冻拳、雷电拳 |
| Lv.40 | 圆环护盾 | 战斗中周期性给予玩家短暂减伤护盾 |
| Lv.50 | 裂隙感应 | 提示附近圆环裂隙 |
| Lv.60 | 双环召唤 | 允许两个宝可梦同时出战 |
| Lv.70 | 异界支援 | 战斗中随机召唤短暂支援攻击 |
| Lv.80 | 神兽共鸣 | 解锁神兽投影技能 |
| Lv.90 | 天穹之环 | 解锁裂空座类支援技能 |
| Lv.100 | 封印之环 | 大胡帕最终战专属机制 |

## 10. 羁绊等级设计

羁绊等级建议最高 5 级。

它不靠刷怪提升，而是通过剧情和关键事件提升。

| 羁绊等级 | 解锁内容 | 剧情含义 |
|---|---|---|
| 羁绊 1 | 胡帕跟随玩家 | 还很害怕 |
| 羁绊 2 | 解锁宝可梦召唤界面 | 开始信任玩家 |
| 羁绊 3 | 解锁圆环裂隙任务 | 主动求助 |
| 羁绊 4 | 解锁神兽投影 | 愿意面对追击 |
| 羁绊 5 | 解锁大胡帕最终战 | 不再逃避 |

羁绊经验来源：

- 首次击败关键 Boss；
- 完成胡帕任务；
- 击败大胡帕幻影；
- 制作关键突破道具；
- 完成大型圆环裂隙事件。

## 11. 推荐体验流程

### 11.1 开局阶段

玩家捡到“破碎圆环”，召唤 Lv.1 胡帕。  
胡帕只能偶尔发射小能量球，并帮助玩家召唤一只弱小宝可梦。

### 11.2 前期阶段

玩家打怪，胡帕升到 Lv.10。  
需要制作“粗糙圆环核心”突破到 Lv.20。  
突破后解锁基础宝可梦召唤池。

### 11.3 肉山前中期

击败腐化/猩红 Boss 后，胡帕意识到泰拉瑞亚世界也存在邪神污染。  
胡帕突破到 Lv.30-40，解锁幽灵、毒、恶属性相关宝可梦。

### 11.4 肉山后阶段

击败血肉墙后，世界边界被撕裂。  
大胡帕感应到小胡帕的位置。  
世界开始出现圆环裂隙事件。

### 11.5 困难模式阶段

胡帕通过困难矿石继续修复圆环。  
机械三王分别提供关键羁绊经验。  
胡帕 Lv.60-70 后可召唤强力宝可梦。

### 11.6 后期阶段

世纪之花后，胡帕开始掌握神兽投影。  
Lv.90 解锁裂空座短暂支援。

### 11.7 终局阶段

月亮领主被击败后，胡帕圆环完全暴露。  
大胡帕正式降临。  
只有胡帕达到 Lv.100 且羁绊等级达到 5，才能开启最终战。

## 12. 最终机制总结

推荐最终设计如下：

```text
胡帕拥有“圆环同步等级”和“羁绊等级”。

圆环同步等级：
- 通过打怪、Boss、圆环裂隙事件提升；
- 最高 Lv.100；
- 每 10 级需要突破；
- 决定召唤槽、宝可梦池、辅助技能和神兽投影。

羁绊等级：
- 通过主线、Boss 首杀和胡帕任务提升；
- 最高 5 级；
- 决定剧情推进、大胡帕追击事件和最终 Boss 解锁。
```

一句话概括：

> 刷怪让胡帕变强，挖矿让胡帕突破，Boss 让剧情推进，羁绊让胡帕敢于面对大胡帕。

## 13. 设计优势

这套机制有几个优点：

1. 与泰拉瑞亚原版流程高度绑定；
2. 不会让玩家脱离挖矿、打 Boss、探索的核心玩法；
3. 胡帕不是普通宠物，而是整个 Mod 的成长核心；
4. 宝可梦召唤有合理剧情解释；
5. 大胡帕最终战有明确铺垫；
6. 经验、突破、剧情、召唤系统可以自然连接。

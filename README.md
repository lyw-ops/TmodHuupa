# Hoopa Across Terraria

Early tModLoader mod prototype for a Hoopa-themed Terraria adventure.

## Current Prototype

- `Hoopa Summon Staff`: craftable summon weapon.
- Swing once to summon one Hoopa minion.
- Hoopa currently follows the player and gains Ring Sync experience while summoned.
- Hoopa fires a simple small ring bolt at nearby enemies.
- Battle skills and breakthrough items will be added later.

## Build Notes

This repository imports the local Steam tModLoader `tMLMod.targets` on this machine. For normal tModLoader development, place or symlink this folder under your tModLoader `ModSources` directory and build it from tModLoader's Develop Mods menu.
